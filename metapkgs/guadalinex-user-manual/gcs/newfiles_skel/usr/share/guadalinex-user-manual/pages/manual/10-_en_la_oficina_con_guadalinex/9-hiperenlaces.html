9-Hiperenlaces

Antes de nada, vamos a intentar explicarte qué es esto del hiperenlace, es más fácil de lo que crees, sólo que tiene un nombre un poco raro.

El hiperenlace es un recurso que se utiliza para relacionar o enlazar partes de un documento con otro punto concreto del mismo o con otro documento ajeno que esté relacionado.

El procesador de textos permite realizar cuatro tipos de enlaces a: dirección web de internet, dirección de correo electrónico, documento de texto que ya existe y documento de texto nuevo.

Para crear un hiperenlace dentro de tu texto tienes que acceder al menú Insertar de la barra de menús y elegir la opción Hiperenlace.

En la ventana que se te abre encontrarás un panel lateral con cuatro opciones:

1- Internet.

2- Correo y noticias.

3- Documento.

4- Nuevo documento.

OpenOffice.org Procesador de textos utiliza la palabra hiperenlace para definir este tipo de recursos, pero también lo encontrarás definido bajo términos como enlace, vínculo, hipervínculo o link. Usa el que sea más fácil para ti. Todos valen.
