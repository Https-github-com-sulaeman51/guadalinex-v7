Guadalinex v7 Manual de usuario

Usuarios

Guadalinex cuenta con la gran ventaja, al igual que otros sistemas operativos basados en Linux, de ser un sistema multiusuario, es decir, que pueden manejar el mismo ordenador varias personas. Puedes tener un solo ordenador en casa que maneje toda la familia.

Por eso es tan importante definir una serie de cuestiones en relación a cada uno de los usuarios o grupos de usuarios que vayan a utilizar el ordenador. Estas cuestiones hacen referencia a:

  • Cuentas de usuario: cada usuario del sistema debe tener una cuenta propia, con su nombre de usuario y su contraseña, para iniciar sesiones independientes de trabajo en el mismo ordenador.
  • Sesiones de trabajo: pueden ser personalizadas según el perfil de cada usuario.
  • Perfiles de usuario: tipos de usuarios según los privilegios que se les hayan concedido.
  • Privilegios de usuario: especifican que tareas puede hacer cada usuario.

Cada usuario, para asegurar la privacidad dentro del sistema, además de tener una cuenta propia debe pertenecer a un grupo de usuarios y ser dueño de sus propios archivos.

Todos los usuarios dentro de la estructura de árbol del sistema tienen un espacio de disco duro asignado para guardar sus archivos, y este espacio es el directorio /home.

Perfiles de usuarios

Vamos a hablarte ahora de los perfiles de usuario según los privilegios que tengan, y son:

  • Administrador: se corresponde con aquel usuario o grupo de usuarios que pueden hacer tareas de administración y tienen control sobre el sistema.

El administrador puede convertirse en usuario Root, esto quiere decir, que cuando va a realizar alguna tarea administrativa el sistema le pide su contraseña y accede como usuario root para realizar cambios como instalación de programas, creación de usuarios, asignar contraseñas, configurar partes del sistema, etc…

Quizás te preguntes por qué el administrador no entra al sistema como usuario root y así se evita tener que estar escribiendo la contraseña cada vez que quiera realizar una tarea de administración, pues la explicación es que el usuario root se crea por motivos de seguridad, si el administrador siempre fuera root podría cambiar partes del sistema o borrar archivos imprescindibles sin darse cuenta.

El administrador como root es el único que puede crear nuevos usuarios o grupos de usuarios, asignarles contraseñas, eliminar esas contraseñas, instalar programas, asignar permisos a directorios y archivos, etc…

  • Usuario del escritorio: aquel usuario o grupos de usuarios que pueden trabajar en su escritorio personal con todos los privilegios menos el de administración del sistema.
  • Usuario personalizado: este tipo de usuario o grupo de usuarios tiene cada uno de los privilegios que tu le quieras dar de forma personalizada. Podría ser un usuario sin privilegios o con privilegios muy limitados, sólo para realizar operaciones muy básicas, por ejemplo, para usar tu ordenador durante un período corto de tiempo, o podría ser un usuario que gozara de todos los privilegios disponibles.

Crear usuarios

Para crear un usuario nuevo en tu sistema debes hacer clic en el panel superior del escritorio en el menú Sistema→Administración→Usuarios y Grupos.

Haz clic en el botón Añadir situado debajo de la lista de usuarios. Lo normal es que el sistema te pida tu contraseña para convertirte en usuario root y poder crear usuarios.

Una vez que has tecleado tu contraseña accedes a una ventana de título Crear un usuario nuevo donde debes especificar el Nombre y Nombre corto del usuario que quieres añadir. Después haz clic en el botón Aceptar.

A continuación, te aparecerá otra ventana de título Cambiar la contraseña del usuario donde puedes generar la contraseña del nuevo usuario de dos maneras: a mano, escribiendo la contraseña que tu elijas para el usuario; y de forma aleatoria, donde es el sistema el que te da una contraseña al azar para tu usuario. Te recomendamos que generes la contraseña a mano, siempre será más fácil de recordar que la que genere el sistema. Termina haciendo clic en el botón Aceptar para guardar los datos.

En este momento el sistema ha creado un nuevo usuario al que ha adjudicado por defecto un perfil de usuario de escritorio. Si quieres que ese nuevo usuario tenga otro tipo de perfil (administrador o usuario personalizado) tendrás que cambiarlo de forma manual. Para ello, selecciona el usuario en la lista de la izquierda y haz clic en el botón Cambiar del apartado Tipo de cuenta situado en la parte derecha de la ventana.

En la ventana que se abre te encuentras con las tres opciones de perfiles de usuario: personalizado, administrador y usuario de escritorio. Para crear un tipo de perfil de usuario personalizado tienes que definir antes los privilegios que adjudicas a ese usuario en concreto. Para ello, haz clic en el botón Ajustes avanzados, entra en la pestaña Privilegios del usuario de la ventana que se te abre, y marca uno a uno aquellos privilegios que le vas a conceder a este usuario. Por último, haz clic en el botón Aceptar.

Modificar y borrar usuarios

Para modificar la información relativa a un usuario tienes que seleccionarlo en la lista de usuarios que aparece en la ventana para a continuación hacer clic en el botón Cambiar de las opciones que te aparecen en la parte derecha de la ventana. Puedes modificar el nombre del usuario, el tipo de cuenta o de perfil y su contraseña. Para modificar su información de contacto, los privilegios, desactivar su cuenta, etc… tienes que acceder al botón Ajustes avanzados.

Y para borrar algún usuario tienes que seleccionarlo en el listado y después hacer clic en el botón Eliminar. El sistema además te pregunta si estás seguro de borrar la cuenta de ese usuario para así evitar que se elimine algún usuario por error.

Tienes más información sobre los tipos de perfiles de usuarios en la sección Pantalla de acceso al sistema de este manual.
